# Advent of Code 2025 - Day 05

This project contains the solution for Advent of Code 2025, Day 05.

## Setup

1.  **Create and activate the virtual environment:**
    ```bash
    uv venv
    source .venv/bin/activate
    ```

2.  **Install dependencies:**
    ```bash
    uv pip install -e '.[dev]'
    ```

## Usage

### Running the Solution

Place your puzzle input into `input.txt`.
To run the main solution:

```bash
python -m main
```

### Running Tests

Place your example input into `example.txt`.
To run the tests:

```bash
pytest
```

### Formatting and Linting

To automatically format your code and check for linting errors:

```bash
ruff check . --fix
```

### Type Checking

To run the static type checker:

```bash
mypy .
```
